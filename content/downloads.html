Download Ibis

The various subprojects of Ibis are available for download. Ibis is released under a BSD style license

Communication library and Programming models

Ibis 1.4 is now available. The software is currently distributed as source code release and as binary release. The distribution includes several Ibis implementations, and implementations of the Satin, RMI, MPJ, and GMI progamming models. Many example programs for these models are distributed separately.
Version Date New in this release Download link
1.4 15 September 2006 This release contains an almost completely rewritten version of Satin. It also fixes various problems with release 1.3. It is also the first release that has the applications in a separate file, and is available as a binary release. Download page
1.3 14 Februari 2006 This release contains the following additions:
  • Shared objects for Satin. This is a replacement (and major improvement) of the TupleSpace, which was a hack, really. The TupleSpace is still there, but it is deprecated and will probably not be there anymore in the next Ibis release
  • Improved/rewritten connection setup code
  • Fixed various scalability issues, mostly in the nameserver. Ibis now scales up to at least a thousand nodes
 Download page
1.2.1 2 December 2005 Maintainance release Download via FTP
1.2 20 September 2005 Adds MPJ/Ibis, a pure Java implementation of he MPJ programming interface that has been defined by the Java Grande forum to provide MPI-like message passing for Java applications. Download via FTP
1.1 13 June 2005 Maintainance release Download via FTP
1.0 24 November 2004 First Ibis release. includes the IPL, and the following implementations of it:
  • TcpIbis, which is an implementation on top of TCP
  • NioIbis, which is an implementation on top of Java NIO
  • MessagePassing Ibis, which is an implementation on top of Panda and MPI, but that is not really tested
  • NetIbis, which can do TCP, UDP, GM, and others
This release also contains Ibis RMI, which is a (incomplete) re-implementation of Java RMI, Satin, which is an implementation of the divide-and-conquer programming model, and GMI, which is an RMI enhanced with group communication. 		Download via FTP
